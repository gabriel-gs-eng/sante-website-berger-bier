# Changing history folder

In "Sobra a Berger Bier" folder, you will replace the current picture with a video, but with different layouts on mobile and desktop:

# On destkop

Replace current picture with @Media/History/history_background_preferred_no_audio.webm. Make the video height be the same of the red frame. Insert autoplay, mute and infinite loop flags.

# On mobile

Remove the picture section and add the @Media/History/history_background_preferred_no_audio.webm as background video, which text will be in front of. Add an oppacity of 35% for video. Insert autoplay, mute and infinite loop flags.
